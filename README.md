# Monitoramento de Servidor

Script de monitoramento de infraestrutura Linux com interface de menu interativa.

## 📋 Funcionalidades

- ✅ Verificar conexão com internet
- ✅ Monitorar uso de disco (alertas acima de 80%)
- ✅ Monitorar uso de memória (alertas acima de 10GB)
- ✅ Monitorar processos SSH
- ✅ Monitorar CPU (uptime)
- ✅ Gerar alertas automáticos
- ✅ Gerar relatórios

## 🚀 Como usar

```bash
menu
```

## 📁 Estrutura

- `scripts/menu` - Interface principal
- `scripts/monitoramento.sh` - Script de coleta de dados
- `logs/` - Logs de execução e alertas
- `relatorios/` - Relatórios gerados
